# mk - 业务模块

## 依赖

<details>
  <summary>常用模块（基础模块）common</summary>

1. `common`

``` xml
<dependency>
  <groupId>com.github.xujiaji.mk</groupId>
  <artifactId>common</artifactId>
  <version>${revision}</version>
</dependency>
```

2. `common-admin`常用模块管理控制

``` xml
<dependency>
  <groupId>com.github.xujiaji.mk</groupId>
  <artifactId>common-admin</artifactId>
  <version>${revision}</version>
</dependency>
```
</details>
