# mk - 业务模块

## 依赖

<details>
  <summary>常用模块（基础模块）`common`</summary>

1. `mk-common`

``` xml
<dependency>
  <groupId>com.github.xujiaji.mk</groupId>
  <artifactId>mk-common</artifactId>
  <version>0.0.2</version>
</dependency>
```

2. `mk-common-admin`常用模块管理控制

``` xml
<dependency>
  <groupId>com.github.xujiaji.mk</groupId>
  <artifactId>mk-common-admin</artifactId>
  <version>0.0.2</version>
</dependency>
```
</details>
